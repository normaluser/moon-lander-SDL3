# Moon lander

## a 2D game of gravity using SDL3 / Pascal

This is my attempt to bring "moon lander" back to life.  
This version of Moonlander is almost rewritten from scratch in SDL3.

# Dependencies

Windows and Linux:

- sdl3

- sdl3-image

- sdl3-mixer

- JsonTools

- Lazarus / Freepascal

- Pascal SDL3 Header files 

   [https://github.com/sechshelme/Lazarus-SDL-3.0-Packages-and-Examples](https://github.com/sechshelme/Lazarus-SDL-3.0-Packages-and-Examples)
   
   [https://github.com/PascalGameDevelopment/SDL3-for-Pascal](https://github.com/PascalGameDevelopment/SDL3-for-Pascal)

a good installation guide for SDL3 can be found at: [https://www.freepascal-meets-sdl.net/](https://www.freepascal-meets-sdl.net/)

# Why ?

I want to learn how to use SDL3 to get easy acess to graphic, sound and input.

# Controls:

- ESC -\> show menue to change game options

- Arrow Keys -\> controll the ship

- Q -\> quit the game

- P -\> pause the game

- A -\> control (toggle) the autopilot

- F -\> control (toggle) the flightdirector
